Описание бота German Helper:

Я сделал бота-помощника для немецкого языка. Его главная цель, помочь людям,
которые хотят выучить немецкий- выучить немецкий. В самом боте присутстувует 4 главных функции, а именно:


1. Информация про темы.
У Пользователя есть возможность получить информацию про темы, аудио, фото и описание.В зависимости от версии,
будет отличаться количество тем которые он может выбрать.

2. Рабочие листы.
У пользователя также есть возможность получить документ, который в будущем можно распечатать.
 Также в зависимости от версии, будет отличаться количество доступных тем

3. Игра "Как на Deutsch?".
Суть игры, помочь подтянуть словарный запас. У пользователя будет возможность выбрать 3 уровня,
которые отличаються по сложности. За каждый правильный ответ пользователь получает +1 токен.

4. Пройти тест.
Также в боте можно будет пройти тест, с которым пользователь может узнать свой уровень нмецкого.
